# Facial-Expression-Detection-Pytorch
Yet another wheel for Facial Expression Detection, based on MTCNN and VGG. Trained on FER2013 and jaffe datasets.

Star this repo if it is awesome!

## Dependencies ##
- python 3.7
- pytorch 1.0
- torchvision
- opencv
- matplotlib (for GUI)
